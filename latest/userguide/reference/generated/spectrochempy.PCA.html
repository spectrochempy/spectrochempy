spectrochempy.PCA¶

class PCA(dataset, centered=True, standardized=False, scaled=False)[source]¶

Principal Component Analysis

This class performs a Principal Component Analysis of a NDDataset, i.e., a linear dimensionality reduction using Singular Value Decomposition (SVD) of the data to perform its projection to a lower dimensional space.

The reduction of a dataset \(X\) with shape (M,`N`) is achieved using the decomposition : \(X = S.L^T\), where \(S\) is the score’s matrix with shape (M, n_pc) and \(L^T\) is the transposed loading’s matrix with shape (n_pc, N).

If the dataset X contains masked values, these values are silently ignored in the calculation.

Parameters

  • dataset (|NDDataset| object) – The input dataset has shape (M, N). M is the number of observations (for examples a series of IR spectra) while N is the number of features (for example the wavenumbers measured in each IR spectrum).

  • centered (bool, optional, default:True) – If True the data are centered around the mean values: \(X' = X - mean(X)\).

  • standardized (bool, optional, default:False) – If True the data are scaled to unit standard deviation: \(X' = X / \sigma\).

  • scaled (bool, optional, default:False) – If True the data are scaled in the interval [0-1]: \(X' = (X - min(X)) / (max(X)-min(X))\)

Methods

printev

Prints figures of merit : eigenvalues and explained variance for the first n_pc PS’s.

reconstruct

Transform data back to the original space using the given number of PC’s.

reduce

Apply a dimensionality reduction to the X dataset of shape [M, N].

scoreplot

2D or 3D scoreplot of samples.

screeplot

Scree plot of explained variance + cumulative variance by PCA.
