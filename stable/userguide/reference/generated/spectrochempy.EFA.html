spectrochempy.EFA¶

class EFA(dataset)[source]¶

Evolving Factor Analysis.

Performs an Evolving Factor Analysis (forward and reverse) of the input NDDataset.

Parameters

dataset (|NDDataset| object) – The input dataset has shape (M, N). M is the number of observations (for examples a series of IR spectra) while N is the number of features (for example the wavenumbers measured in each IR spectrum).

Methods

get_conc

Computes abstract concentration profile (first in - first out)

Attributes

b_ev

NDDataset - Eigenvalues for the backward analysis

cutoff

float - cutoff value

f_ev

NDDataset - Eigenvalues for the forward analysis
