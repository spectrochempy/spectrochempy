spectrochempy.EFA¶

class EFA(**kwargs)[source]¶

Evolving Factor Analysis.

Perform an Evolving Factor Analysis (forward and reverse) of the input NDDataset .

Parameters

dataset (|NDDataset| object) – The input dataset has shape (M, N). M is the number of observations (for example a series of IR spectra) while N is the number of features (for example the wavenumbers measured in each IR spectrum).

Methods

EFA.get_conc([n_pc])

Computes abstract concentration profile (first in - first out).

Attributes

EFA.b_ev

Eigenvalues for the backward analysis (NDDataset).

EFA.cutoff

Cutoff value (float).
