# Maintainer Documentation

This directory contains operational procedures for SpectroChemPy releases and
recovery.

Shared stable maintainer knowledge now lives in
[`../maintainers/`](README.md).

That public maintainer subset currently includes:

- stable RFCs;
- maintained architecture references;
- stable governance references;
- selected historical decision documents that remain useful for context.

For contributors and plugin authors, see the developer documentation in
[`docs/sources/devguide/`](../docs/sources/devguide/).
