# Governance References

This directory contains the small shared subset of maintainer-governance
documents that are stable enough to live in the main `spectrochempy`
repository.

Use these documents for shared process boundaries that matter beyond one
maintainer's private workflow.

## Documents

- [`document-scope-and-promotion.md`](document-scope-and-promotion.md)
  Shared boundary for what belongs in the public maintainer corpus, what stays
  private, and how private material should be promoted.
- [`official-plugin-documentation-standard.md`](official-plugin-documentation-standard.md)
  Shared documentation baseline for official SpectroChemPy plugins.
