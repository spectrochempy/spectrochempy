NMR processing (plugin-based)
-----------------------------

.. note::

    These examples require the official ``spectrochempy-nmr`` plugin.

    Install it with:

    .. code-block:: bash

        pip install spectrochempy[nmr]

    See :ref:`plugins` for more information.
