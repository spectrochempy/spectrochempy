# spectrochempy-myplugin

A plugin for [SpectroChemPy](https://github.com/spectrochempy/spectrochempy).

## Installation

```bash
pip install spectrochempy-myplugin
```

## Development

```bash
git clone <repo-url>
cd spectrochempy-myplugin
pip install -e ".[dev]"
```

Run tests:

```bash
python -m pytest tests/ -v
```
