spectrochempy.NDDataset.save¶

NDDataset.save(**kwargs)[source]¶

Save the current object in SpectroChemPy format.

Default extension is *.scp for NDDataset’s and *.pscp for Project’s.

Parameters

**kwargs – Optional keyword parameters (see Other Parameters).

Other Parameters

confirm (bool)

See also

save_as

Save current object with a different name and/or directory.

write

Export current object to different format.

Examples

Read some data from an OMNIC file >>> nd = scp.read_omnic(‘wodger.spg’) >>> assert nd.name == ‘wodger’

Write it in SpectroChemPy format (.scp) (return a pathlib object) >>> filename = nd.save()

Check the existence of the scp fie >>> assert filename.is_file() >>> assert filename.name == ‘wodger.scp’

Remove this file >>> filename.unlink()
